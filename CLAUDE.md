# Project guide for Claude

The canonical project guide for AI coding assistants — including Claude —
lives in [`AGENTS.md`](./AGENTS.md).

This file exists so that Claude tooling (Claude Code, Cowork, Claude in
IDEs) picks up the guide automatically. The same content is also exposed
to GitHub Copilot via `.github/copilot-instructions.md`. Single source of
truth, multiple discovery paths.

**Always read `AGENTS.md` before writing any code in this repository.**

For Claude-specific extensions — sub-agents, slash commands, skills —
see the directories under `.claude/`.
